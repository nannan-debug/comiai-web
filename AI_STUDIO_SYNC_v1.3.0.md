# ComiAI v1.3.0 同步说明

这份文档用于发给 AI Studio，帮助它将当前 `v1.3.0` 的主要模块和交互同步到另一份项目中。

## 本次版本信息

- 产品名：`ComiAI`
- 仓库名：`comiai-web`
- 版本号：`v1.3.0`

## 主要涉及文件

请优先同步以下文件：

1. `src/App.tsx`
2. `src/ScriptAndAssets.tsx`
3. `src/StoryboardManagement.tsx`
4. `src/StoryboardProduction.tsx`
5. `README.md`
6. `package.json`
7. `package-lock.json`

---

## 一、流程主线调整

### 1. 剧本与设定页先做方案选择与预览

`剧本与设定页` 现在不只是做资产确认，还承担：

- 生成拆分镜方案
- 预览多个方案
- 选定一个采用方案

然后再进入 `分镜管理页`。

### 2. 分镜管理页不再承担“方案对比”

用户在 `剧本与设定页` 先选定方案后：

- 进入 `分镜管理页`
- 页面顶部只显示一个当前初始化模式
- 不再在页面级切换方案
- 但每个分镜卡内部仍可切换图片/视频工作模式

### 3. 分镜制作页支持按素材类型进入不同面板

从 `分镜管理页` 中：

- 点图片素材的“进入精修” -> 进入图片精修面板
- 点视频素材的“进入精修” -> 进入视频精修面板

也就是说，第 3 步 `分镜制作` 现在支持按进入来源切到不同工作台。

---

## 二、`src/App.tsx` 改动

### 1. 分集管理 -> 剧本分集

在“分集管理”里点击 `选择已有剧本` 后：

- 会进入“剧本分集”页
- 用户可以在该页选择其他集内容

### 2. 分镜管理 -> 分镜制作的跳转带面板类型

`App.tsx` 里新增了第 3 步工作台的目标状态传递：

- `image`：进入图片精修工作台
- `video`：进入视频精修工作台

用于承接分镜管理页里不同素材的 hover 操作。

### 3. `StoryboardProduction` 支持初始化模式

`StoryboardProduction` 现在支持从外部传入：

- `initialGlobalMode`
- `initialTaskMode`

这样从分镜管理页点“图片精修”或“视频精修”时，能直接打开对应面板。

---

## 三、`src/ScriptAndAssets.tsx` 改动

这是本次改动最多的页面。

### 1. 左侧剧本区文案调整

- `输入本集剧本` -> `本集剧本`
- 右上角 `导入文件` -> `可编辑`

并且左侧剧本区本身就是可编辑 `textarea`，用户可以直接在此页修改剧本。

### 2. 提示词规则区折叠

左下角提示词和方案规则区改为可展开/收起：

- 收起时只保留轻量入口
- 展开后可设置方案与提示词

减少对剧本区的遮挡。

### 3. 设定资产 tab 改为完整资产目录

右侧 `设定资产` tab 已扩展成三类资产：

- 角色
- 场景
- 道具

每个区块支持：

- 展示 AI 识别资产
- 新建资产入口
- 点击卡片继续设置

并去掉了多余的“资产目录”标题，只保留一条轻量说明。

### 4. 拆分镜预览 tab 前置

右侧新增双 tab：

- `设定资产`
- `拆分镜预览`

在 `剧本与设定页` 就可以先预览分镜方案，不需要等到 `分镜管理页` 再切来切去。

### 5. 拆分镜方案规则

支持两种方案：

- `多参生视频方案`
- `图生视频方案`

方案可以单选或多选生成，然后在预览页中选定采用方案。

### 6. 方案案例数区分

为了体现两种方案拆法不同：

- `多参生视频方案`：3 个分镜
- `图生视频方案`：5 个分镜

### 7. 拆分镜预览表结构调整

预览表现在是表格式工作表，字段包括：

- 镜号
- 状态
- 角色
- 场景
- 道具
- 描述词列
- 台词
- 音效

描述词列按方案不同切换：

- `多参生视频方案`：只展示 `视频描述词`
- `图生视频方案`：只展示 `画面描述词`

并且：

- `备注` 已统一改成 `音效`
- `状态` 字段已补回

### 8. 列宽与视觉优化

预览表做了专门的宽度分配：

- `角色 / 场景 / 道具` 列更窄
- `画面描述词 / 视频描述词` 列更宽
- `台词 / 音效` 列更窄
- 表头与内容对齐

### 9. 角色/场景/道具升级成彩色 tag 输入

这三列不再是纯文本编辑框，而是彩色 tag 输入体验：

- 角色：蓝色 tag
- 场景：绿色 tag
- 道具：紫色 tag

支持：

- 回车新增 tag
- 删除单个 tag
- 更像“资产关联”而不是普通文案输入

### 10. 拆分镜预览支持直接编辑

用户在预览表中可以直接编辑：

- 状态
- 角色
- 场景
- 道具
- 描述词
- 台词
- 音效

### 11. 支持新增/删除分镜

在拆分镜预览中：

- 鼠标悬浮到行间会出现“新增”按钮
- 鼠标悬浮到当前行会出现“删除”按钮
- 删除或插入后，镜号自动重新排序

注意：

- 新增分镜已改成“插入空白分镜”
- 不再复制上一镜内容

### 12. 选定方案后进入分镜管理页

在 `拆分镜预览` 中点 `选定当前方案` 后：

- `下一步：制作分镜表` 才能使用
- 进入分镜管理页时，会根据采用方案决定初始化模式：
  - 多参生视频 -> `direct-video`
  - 图生视频 -> `image-video`

## 四、`src/StoryboardManagement.tsx` 改动

### 1. 顶部只显示一个初始化模式

页面顶部不再显示多方案切换，而是只显示当前采用的初始化模式：

- `多参生视频`
- `图生视频`

但每个镜头卡内部，仍可切换对应工作方式。

### 2. 不同场景案例补全

分镜管理页的分镜卡不再是重复示例，而是不同场景案例，例如：

- 陷阵营地
- 篝火区域
- 营帐通道
- 营地边缘

并且每个案例都带不同的角色/场景/道具组合。

### 3. 两类生成状态都具备完整案例

无论是图片还是视频区域，现在都覆盖了三种状态：

- 未生成
- 生成中
- 已生成

用户点击图片区或视频区，都可以直接触发生成任务。

### 4. 图生视频工作流的右侧为双内容区

如果当前初始化模式是 `图生视频`：

右侧 `预览与生成` 会同时出现两个区域：

- 图片区
- 视频区

更符合“先图后视频”的工作流。

### 5. 多参生视频工作流的右侧更偏单一视频工作流

如果当前初始化模式是 `多参生视频`：

右侧仍保持更偏视频直出的布局。

### 6. 已生成素材 hover 操作

当图片区或视频区已经有生成好的素材时：

鼠标 hover 上去会浮现两个按钮：

- `图片精修` / `视频精修`
- `重新生成`

逻辑为：

- 点 `重新生成`：留在当前页重新触发生成
- 点 `图片精修`：跳到图片精修工作台
- 点 `视频精修`：跳到视频精修工作台

---

## 五、`src/StoryboardProduction.tsx` 改动

### 1. 支持按入口初始化工作台

`StoryboardProduction` 不再只是固定默认状态，而是能根据上一页传参决定初始面板：

- 如果从图片精修进入 -> 直接打开图片工作台
- 如果从视频精修进入 -> 直接打开视频工作台

### 2. 图片与视频工作台继续区分

两种工作台仍分别保留自己的侧栏与配置逻辑：

- 图片工作台：参考内容、画面提示词、补充提示词、模型选择
- 视频工作台：全能参考 / 首尾帧 / 图生视频 等模式与视频生成配置

---

## 六、版本同步建议

如果 AI Studio 要同步这版，建议按下面顺序处理：

1. 先同步 `App.tsx`
2. 再同步 `ScriptAndAssets.tsx`
3. 再同步 `StoryboardManagement.tsx`
4. 最后同步 `StoryboardProduction.tsx`

这样页面间的跳转和状态联动更不容易断。
