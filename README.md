# ComiAI

当前版本：`v1.5.1`

## 更新说明
本次版本更新主要优化了创作流程中的 UI 交互体验与分镜方案流转：

### 1. UI 布局优化
- **风格与尺寸选择器迁移**：将“风格”和“尺寸”选择器从分集剧本页面移动到了“资产绑定面板”和“分镜管理页”的顶部工具栏。
- **流程一致性**：确保在创作的不同阶段，用户都能快速调整全局视觉设定。

### 2. 剧本与设定页增强
- **方案规则折叠**：提示词和方案规则改为折叠配置，减少对剧本输入区的遮挡。
- **拆分镜预览前置**：在“剧本与设定”阶段即可预览不同拆分镜方案，并先选定采用版本。

### 3. 分镜管理页同步
- **初始化模式联动**：分镜管理页会根据上一页选定的方案初始化默认模式。
- **镜头级切换保留**：虽然页面顶部只展示当前初始化模式，但每个分镜仍可独立切换模式。

## 如何运行
1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 构建项目：`npm run build`
